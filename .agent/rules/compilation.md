# Compilation Rules

When you modify any Pascal source files (`.pas`, `.lpr`, `.lpi`, `.dfm`, `.lfm`), you **MUST** run the compilation script to verify your changes.

**Command:**
`compile_win.bat` (located in the project root)

**Instructions:**
1.  **Always compile** after making changes to code.
2.  If the compilation **fails**:
    *   Analyze the error message.
    *   **Fix the error** in the code.
    *   **Re-run** `compile_win.bat` to verify the fix.
    *   **Do not** mark the task as complete until the code compiles successfully.
3.  If the compilation **succeeds**, you may proceed.
